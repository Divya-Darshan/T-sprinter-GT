##  The To-Do Roadmap

### Phase 1: Core Mechanical Movement (Next Steps)
- [ ] **Motor Driver Integration:** Connect the physical motor driver inputs to the D1 Mini pins once the package arrives.
- [ ] **Dual-Motor Configuration:** Wire up the front and rear motors to test the custom counter-rotating layouts.
- [ ] **PWM Speed Controls:** Update the software from a simple On/Off toggle to sliding power values (0% to 100%) so you can control tire slip-angles smoothly while drifting.

### Phase 2: Web Interface & Branding
- [ ] **Clean Domain Name:** Change the network routing from `.local` to a custom product URL (like `http://trueno.drift` or `http://play.trueno`).
- [ ] **Sleek Control UI:** Build a custom HTML/CSS touch interface featuring dual joysticks or control pads instead of a raw text button.
- [ ] **E-Commerce Integration:** Embed an offline product showcase/link inside the vehicle's hosted interface to direct players to your upcoming parts storefront.

### Phase 3: Multi-Car Ecosystem (For Scalability)
- [ ] **Custom Device Naming:** Create a setup mode that allows a user to change their specific car's Wi-Fi name (SSID) and password from the browser.
- [ ] **Memory Persistence:** Save those custom user settings directly into the ESP8266’s permanent flash memory (EEPROM) so it remembers its name after power loss.
- [ ] **Anti-Interference Channels:** Program the cars to automatically scan and pick clear Wi-Fi channels so multiple people can race and drift in the same room without lagging each other out.

### Phase 4: Hardware Reliability & Production Strategy
- [ ] **Power Line Decoupling:** Add capacitors across the power rails to shield the Wi-Fi chip from the electrical noise generated by high-power spinning motors.
- [ ] **Low-Voltage Telemetry:** Track battery life via the analog pin (`A0`) to display a real-time battery gauge on the phone screen and protect the LiPo cell from over-draining.
- [ ] **Monolithic Integrated PCB:** Move away from loose prototyping wires and compress the Lolin D1 Mini chip, charging circuit, and motor driver onto a single custom circuit board to reduce manufacturing costs.
